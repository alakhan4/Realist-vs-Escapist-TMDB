# Trend analysis for Escapist vs Realist movies from 1900–2023

## 🧠 The Hypothesis
> In the last 20 years, "Escapist" genres (Sci-Fi, Fantasy, and Animation) have experienced a significantly steeper growth in Average Popularity compared to "Realist" genres (Drama, History, and Documentary), which have remained stagnant.

## 📊 Dataset
This analysis leverages the **[TMDB Movies Dataset (1M+ Movies)](https://www.kaggle.com/datasets/asaniczka/tmdb-movies-dataset-2023-930k-movies/data)** provided by asaniczka on Kaggle.
